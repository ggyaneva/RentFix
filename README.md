# RentFix

RentFix is a Java-based web application for property rental and maintenance management.
The project is developed as a practical learning project, focused on backend development,
business logic, and clean application structure.

## Tech Stack
- Java
- Spring Boot
- Spring MVC
- Spring Security
- Thymeleaf
- MySQL
- Maven

## Features
- User authentication and authorization
- Role-based access (Admin, Owner, Tenant)
- Property listing and rental management
- Rental contracts handling
- Maintenance request management
- Basic validation and error handling

## Architecture
The application follows a layered architecture:
- Controller layer
- Service layer
- Repository layer
- DTOs for data transfer

The project is designed with separation of concerns and readability in mind.

## Purpose
The main goal of this project is to practice:
- Java and Object-Oriented Programming
- Spring Boot fundamentals
- Working with databases
- Implementing real-world business logic
